## Project Overview

ESP-IDF haptic FOC knob ("superdial") with LVGL touchscreen UI, USB/Bluetooth HID support, and WiFi configuration. Target: ESP32-S3 (240 MHz, 16MB Flash, 8MB PSRAM).

## Build Commands

```bash
idf.py build                    # Build project
idf.py flash                    # Flash to device
idf.py monitor                  # Serial monitor
idf.py build flash monitor      # Build + flash + monitor
idf.py menuconfig               # SDK configuration GUI
idf.py fullclean                # Clean all build artifacts
```

## Architecture

### Task Structure (FreeRTOS)
- **Motor Task** (Priority 24): 1ms FOC control loop - `motor.loopFOC()` + torque calculation
- **Dial Event Task** (Priority 22): Processes knob/button events from queue
- **LVGL Task** (Priority 4): UI rendering with 5ms timer tick

### Event Flow
```
Knob/Button → dial_publish() → Dial_Queue → Event Handler → UI Screen Handler
```

### Key Directories
- `main/dial/` - FOC motor control (BLDCMotor + MT6701 sensor via SimpleFOC)
- `main/display/` - GC9A01 240x240 LCD initialization
- `main/ui/` - LVGL screens (SquareLine Studio generated), 4 screens
- `main/usb_device/` - TinyUSB HID + BLE HID profiles
- `main/wifi/` - WiFi AP configuration + web server
- `main/nvs_data/` - NVS persistent settings
- `main/dial_power/` - Battery ADC + power management
- `components/foc_knob/` - Custom FOC knob haptic modes component

### Hardware Pins
| Function | GPIO |
|----------|------|
| Motor PWM | 15, 16, 17 |
| Motor Sensor (SPI3) | CLK=2, MISO=1, CS=42 |
| Button | 5 |
| LCD (SPI2) | SCLK=12, MOSI=11, DC=14, CS=10, RST=9 |
| LCD Backlight | 13 |
| Battery ADC | 4 |
| Power Control | 18 |

### FOC Knob Modes (in foc_knob_default.c)
0. Unbounded no detents
1. Unbounded fine detents
2. Unbounded strong detents
3. Bounded no detents (0-360°)
4. Bounded fine detents
5. Bounded strong detents
6. On/Off strong detents

### HID Modes
- USB: 4 report IDs (Keyboard, Mouse, Dial, Consumer)
- BLE: 4 HID devices (Keyboard, Mouse, Dial, Media)
- HID always-on after init (USB re-enumeration issues prevent per-screen toggle)

## Key Implementation Notes

- UI generated by SquareLine Studio - edit in Squareline, export to `main/ui/`
- LVGL 8.3.6 with RGB565, byte-swap enabled
- SimpleFOC library for motor control - use FOC's built-in PID (has time-based D/I)
- Partition table supports OTA (dual app partitions: app0/app1, 4.8MB each)
- DevContainer available for Docker-based development with ESP-IDF + QEMU
